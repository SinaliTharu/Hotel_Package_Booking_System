Hotel Package Booking System (Java)

This project was developed as a Level 3 mini project for the module COU3304 – Fundamentals of Programming.

It is a console-based Hotel Package Booking System developed using Java, designed to allow users to select hotel packages, validate inputs, calculate total costs, and complete a booking process with confirmation.

🔹 Features

Display available hotel packages

User registration and detail collection

Package selection (Family, Couple, Single, Day Out)

Room selection (where applicable)

Dynamic total price calculation

Secure card detail validation

Booking confirmation display

🔹 My Individual Contributions

Implemented DisplayPackages() and UserRegister() functions using Java Collections

Applied conditional logic (IF–ELSE) for process control

Implemented exception handling using try-catch blocks

Designed robust input validation (name, phone number, date format, etc.)

Identified and corrected logical/system errors

Integrated team-developed components into a cohesive system

Performed overall debugging and system improvement
